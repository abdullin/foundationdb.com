DROP GROUP∞

Purpose∞

Use the DROP GROUP statement to drop a Table-Group.

Syntax∞

DROP GROUP [ IF EXISTS ] group_name

Parameters∞

IF EXISTS
Do not return an error if the group does not exist.
group_name
The name (optionally schema-qualified) of the group to drop. This is the root table of a Table-Group.

Notes∞

DROP GROUP is a quick way to remove an entire Table-Group, which includes all tables it contains. This method is much more efficient than removing the tables separately as that would require rewriting the entire group for each table that is removed.

Examples∞

Dropping a group:

DROP GROUP customers;
