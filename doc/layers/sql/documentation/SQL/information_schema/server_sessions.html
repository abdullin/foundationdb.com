Sessions tracks the session connections to the server. When a connection is created, the session created to handle the connection is tracked.

The following table provides a description for the different columns:

Name Data Type Description
session_id bigint The unique session id
caller_session_id bigint  
start_time timestamp The start time of the session
server_type varchar(128)  
remote_address varchar(32) The remote address this session connected from.
session_status varchar(32) The top level of processing for the session
query_count bigint  
last_query_executed varchar(1024) The last SQL statement this session has executed.
query_start_time timestamp  
query_end_time timestamp  
query_row_count bigint  
prepared_name varchar(128)
