SQL Layer Developer Reference∞

The following documents provides content and advice to guide you through building your application with the SQL Layer.

Developer Guide∞

Start here

Developer Guide highlights the differences between the SQL Layer and more traditional single node SQL RDBMSs, and explains the basic principles for building an effective application with the SQL Layer.

SQL Layer Client Tools∞

SQL Layer Client Tools Reference details the command-line tools provided for interfacing with the SQL Layer.

Performance Guide∞

Performance Guide will guide you on best practices for getting the best performance out of your SQL Layer deployment.

Application Integration∞

Application Integration covers the supported ORMs, drivers and IDEs, and provides advice on adapting ones that are not officially supported.

SQL API Reference∞

SQL API Reference gives a detailed description of the SQL language used in the SQL Layer, including details on DDL, DML, indexing, query and execution plans.

REST API Reference∞

REST API reference provides a comprehensive list of entry-points and capabilities provided by the REST API.
