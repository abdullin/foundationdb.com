Functions∞

The following functions are supported in the SQL Layer.

Name Description
abs(numeric) Absolute value.
acos(double) Inverse cosine.
acot(double) Inverse cotangent.
adddate(d datetime, n numeric) Add n days to d.
addtime(t time, n numeric) Add n seconds to t.
aes_decrypt(b binary, key string) Decrypt AES encoded bytes b with the key key.
aes_encrypt(s string, key string) Encrypt s with AES using the key key.
and(x boolean, y boolean) Boolean AND (x && y).
ascii(string) Numeric value of the first byte of a string.
asin(double) Inverse sine.
atan(double) Inverse tangent.
atan2(x double, y double) Inverse tangent of x / y.
avg(numeric) (Aggregate) Average of all input values.
bit_and(integer) (Aggregate) Bitwise AND all input values.
bit_count(integer) Count of the number of bits set in an integer.
bit_length(string) Length of s in bits (8 bits per byte, character set sensitive).
bit_or(integer) (Aggregate) Bitwise OR all input values.
bit_xor(integer) (Aggregate) Bitwise XOR all input values.
bitand(x integer, y integer) Bitwise AND (x & y).
bitnot(integer) Bitwise NOT (~ x).
bitor(x integer, y integer) Bitwise OR (x | y).
bitxor(x integer, y integer) Bitwise XOR (x ^ y).
blike(s string, pattern string[, escape string]) Match pattern against s case-sensitively.
ceil(numeric) Smallest integer not less than x.
chr(integer) Generate a string containing the Unicode character for the given codepoint.
char_length(string) Length of string in code points.
coalesce(p1 any, p2 any, ...) First non-NULL argument.
column_type_string(schema string, table string, column string) String representation of a column’s data type.
concat_ws(separator string, s1 string, s2 string, ...) Combine multiple strings together into one with separator between them.
concat(s1 string, s2 string, ...) Combine multiple strings into one. (alias for concatenate)
concatenate(s1 string, s2 string, ...) Combine multiple strings into one.
conv(x integer, from_base integer, to_base integer) Convert integers from one base to another.
convert_tz(x datetime, from_tz string, to_tz string) Convert datetimes from first one timezone to another.
cos(double) Cosine.
cosh(double) Hyperbolic cosine.
cot(double) Cotangent.
coth(double) Hyperbolic cotangent.
count(any) (Aggregate) Count the number of input elements.
create_random_guid() Generate a type 4 (pseudo-randomly generated) GUID value.
crc32(string) CRC32 value of a string.
curdate Current date. (alias for current_date)
current_date Current date.
current_schema Current schema.
current_session_id() Session ID of the current connection.
current_setting(string) Value of the given property.
current_time Current time.
current_timestamp Current timestamp.
current_user User name of the current connection.
currval(schema string, sequence string) Current value of the given sequence.
curtime Current time. (alias for current_time)
date(datetime) Extract the date portion of a datetime.
date_add(x datetime, i interval) Add an interval to a datetime.
date_format(d date, format string) Format a date as the given string.
date_sub(d datetime, i interval) Subtract an interval from a datetime.
datediff(x datetime, y datetime) Number of days between two days.
datetime(datetime) Extract datetime.
day(datetime) Day from a date.
dayname(date) Name of the weekday.
dayofmonth(datetime) Day from a date. (alias for day)
dayofweek(datetime) Day of the week in range 1 to 7.
dayofyear(datetime) Day of the year in range 1 to 366.
degrees(double) Convert radians to degrees.
describe_expression(expression) String representation of the return type for e.
distance_lat_lon(lat_x numeric, lon_x numeric, lat_y numeric, lon_y numeric) Distance between two latitude and longitude pairs.
div(x numeric, y numeric) Integer (truncating) division of x / y.
divide(x numeric, y numeric) Divide x by y (x / y).
elt(i integer, s1 string, s2 string, ...) Return the string at index i.
exp(double) Raise e (base of natural logarithm) to the argument.
export_set(i integer, on string, off string[, sep string[, num_bits integer]]) String representation of i where on is used for 1, off is used for 0, sep is used for the separator and num_bits are printed.
field(needle string, s1 string, s2 string, ...) Index of the first occurrence of needle or 0 if not found.
first(any) (Aggregate) First non-NULL of all input values.
floor(double) Largest integer not greater than x.
from_base64(string) Decode a base64 encoded string.
from_days(integer) Convert a number of days (since year zero) to a datetime.
from_unixtime(time integer, format string) Format a UNIX Timestamp into a string.
full_text_search(column identifier, s string) Search for s (Lucene Query Syntax) in column. Requires a Full Text Index.
group_concat(string) (Aggregate) Combine all input values into a single output string.
hex(string) Convert each byte of s into a 2-digit hexadecimal number.
hour(datetime) Extract the hour from a datetime.
identity_sequence(schema string, table string, column string) Name of the sequence associated with the given column. (alias for serial_sequence)
if(x boolean, a any, b any) Output a if x is true, otherwise b.
ifnull(x any, y any) Output y if x is NULL, otherwise x.
ilike(s string, pattern string[, escape string]) Match pattern against s case-insensitively.
inet_aton(string) Numeric value of an IPv4 address.
isfalse(boolean) Check if false.
iregex(s string, pattern string, [options string]) Find if pattern regular expression is in s string, using options for regex matching, and case-insensitive match.
isnull(any) Check if NULL.
istrue(boolean) Check if true.
isunknown(boolean) Check if unknown (i.e. null).
last_day(date) Last day of the month for month of a date.
left(s string, n integer) First n characters from a string.
leftshift(i integer, n integer) Bit shift i n places to the left (i << n).
ln(double) Natural (base-e) logarithm.
locate(substr string, s string[, offset integer]) Find the location of the first occurence of substr in s starting at offset.
localtime Current timestamp. (alias for current_timestamp)
localtimestamp Current timestamp. (alias for current_timestamp)
log(double) Natural (base-e) logarithm. (alias for ln)
log10(double) Base-10 logarithm.
log2(double) Base-2 logarithm.
lower(string) Lowercase a string.
lpad(s string, len integer, pad string) Pad the right side of s to a length of len using pad.
ltrim(string) Remove spaces from the left side of s.
maketime(hour integer, minute integer, second integer) Create a time value.
max(any) (Aggregate) Maximum of all input values.
md5(string) MD5 hash.
mid(s string, offset integer, len integer) Substring s of length len starting at offset.
min(any) (Aggregate) Minimum of all input values.
minus(x numeric, y numeric) Subtract y from x (x - y).
minute(datetime) Extract minute value.
mod(x numeric, y numeric) Remainder of x divided by y (x % y).
month(datetime) Extract month value.
monthname(date) Name of the month from a date.
nextval(schema string, sequence string) Next value for the given sequence.
not(boolean) Negate (invert) the given value.
now Current timestamp. (alias for current_timestamp)
nullif(x any, y any) NULL if x equals y, otherwise x.
octet_length(string) Length of s in bytes (character set sensitive).
or(x boolean, y boolean) Boolean OR (x || y).
period_add(period integer, n integer) Add the n months to period (period format is YYYMM or YYMM).
period_diff(x integer, x integer) Number of months between two periods (period format is YYYYMM or YYMM).
pi() Value of π (pi).
plus(x numeric, y numeric) Add x and y (x + y).
position(substr string, s string[, offset integer]) Find the location of the first occurence of substr in s starting at offset. (alias for locate)
pow(x numeric, y numeric) Raise x to the power y (x ^ y).
quarter(date) Quarter (range 1 to 4) for a date.
radians(numeric) Convert degrees to radians.
rand([integer]) Generate a random number the range [0, 1). Starting value is determined by seed.
regex(s string, pattern string, [options string]) Find if pattern regular expression is in s string, using options for regex matching.
repeat(s string, n integer) Concatenate s n many times.
replace(s string, from string, to string) Replace all occurences of from in s with to.
reverse(string) Reverse the input string.
right(s string, n integer) Last n characters from s.
rightshift(i integer, n integer) Bit shift i to the right n times (i >> n).
round(n integer[, d integer]) Round n to d decimal places.
rtrim(string) Remove spaces from the left side of s.
sec_to_time(integer) Create a time value from a number of seconds.
second(datetime) Extract seconds value from a datetime.
serial_sequence(schema string, table string, column string) Name of the sequence associated with the given column.
session_user User name of the current connection. (alias for current_user)
sign(numeric) Sign value (-1, 0 or 1) of a number.
sin(double) Sine.
sinh(double) Hyperbolic sine.
sleep(integer) Sleep (suspend execution) for i milliseconds.
space(integer) Create a string consisting of n spaces (‘ ‘).
sqrt(double) Square-root.
stddev_pop(double) (Aggregate) Population standard deviation of all input values.
stddev_samp(double) (Aggregate) Sample standard deviation of all input values.
strcmp(x string, y string) Return -1, 0 or 1 if x is less, equal or greater than y.
subdate(d datetime, i integer) Subtract i days from d.
substr(s string, offset integer[, len integer]) Return len characters from s starting at offset. (alias for substring)
substring(s string, offset integer[, len integer]) Return len characters from s starting at offset.
substring_index(s string, delimiter string, n integer) A string of the characters before the nth occurrences of delimiter.
subtime(x time, n integer) Subtract n seconds from x.
sum(numeric) (Aggregate) Sum all input values.
sysdate() Current datetime.
system_user() User name the SQL Layer is running as.
tan(double) Tangent.
tanh(double) Hyperbolic tangent.
time(datetime) Extract time from datetime.
time_add(t time, n integer) Add n seconds to t.
time_to_sec(time) Convert time to a number of seconds.
timediff(x time, y time) Subtract y from x (x - y).
times(x numeric, y numeric) Multiply x by y (x * y).
timestamp(datetime) Extract datetime. (alias for datetime)
timestampadd(n integer, interval interval, d datetime) Add n interval to d.
timestampdiff(n integer, interval interval, d datetime) Subtract n interval from ts.
to_base64(binary) Encoding b as a base64 string.
to_days(datetime) Number of days, since year 0, for a datetime.
to_seconds(datetime) Number of seconds (since year 0) for a dateime.
trim(string) Remove leading and trailing spaces.
truncate(n numeric, d integer) Truncate n to d decimal places.
unhex(string) Interpret s as a hexadecimal string and return a binary value from each pair.
unicode(string) First Unicode code point in a string.
unix_timestamp([datetime]) UNIX Timestamp for a datetime.
upper(string) Uppercase a string.
var_pop(double) (Aggregate) Population variance of all input values (square of the population standard deviation).
var_samp(double) (Aggregate) Sample variance of all input values (square of the sample standard deviation).
version() SQL Layer version string.
week(d datetime[, mode integer]) Week number of a date.
weekday(date) Weekday for a date.
weekofyear(date) Week number of a date (equivalent to WEEK(d, 3)).
xor(x boolean, y boolean) Boolean XOR (x ^ y).
year(date) Year of a date.
yearweek(d date[, mode integer]) Year and week (as YYYYWW) for a date.
znear(lat_x numeric, lon_x numeric, lat_y numeric, lon_y numeric) Z-value (nearness estimate) for lat_x/lon_x compared to lat_y/lon_y (requires spatial index).
