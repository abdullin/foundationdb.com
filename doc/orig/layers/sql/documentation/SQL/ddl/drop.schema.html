DROP SCHEMA∞

Purpose∞

Use the DROP SCHEMA statement to drop a schema.

Syntax∞

DROP SCHEMA [ IF EXISTS ] schema_name [ { CASCADE | RESTRICT } ]

Parameters∞

IF EXISTS
Does not return an error if the schema does not exist.
schema_name
The name of the schema to be dropped.
CASCADE
Also drop all objects present in the schema.
RESTRICT
Drops the schema only when no objects are present in the schema. This is the default.

Examples∞

Dropping a schema:

DROP SCHEMA schema_name CASCADE;
